Система моделирования движения транспорта на автодороге (в тоннеле / на автостраде)

Настройка параметров дороги

Для настройки параметров пользователю необходимо указать тип дороги, а также количество направлений и полос дороги

Вернуться к оглавлению

Выбрать тип дороги

Пользователь должен выбрать тип дороги: автострада или тоннель.При выборе типа тоннель, пользователь должен задать значение светофорной фазы.

Указать количество направлений

Пользователь должен указать количество направление дороги от 1 до 2.

Указать количество полос

Пользователь должен указать количество полос дороги от 1 до 3.

Вернуться к оглавлению

Настройка транспортного потока

После настройки параметров дороги, пользователь должен настроить параметры транспортного потока. Пользователь может выбрать детеминированный или случайный закон распределения, указав необходимые параметры.

Выбор типа транспортного потока.

Детерминированный

Для настройки типа дороги "детерминированный", Вам необходимо указать интервал появления автомобилей и их скорость.

Случайный

Выбирая этот параметр, Вам необходимо указать желаемый закон распределения, по которому будет строиться движение транспортного потока.

Нормальное распределение

Чтобы смоделировать транспортный поток по нормальному распределению, Вам нужно указать значение диисспресии и математического ожидания.

Равномерное распределение

Чтобы смоделировать транспортный поток по равномерному распределению, Вам нужно указать левую и правую границы задаваемого интервала.

Показательное распределение

Чтобы смоделировать транспортный поток по показательному распределению, Вам нужно указать значение интенсивности.
